## Exercício 2

Para declarar uma variável em Java, a sintática é semelhante a C++:
```
tipo nome_da_variavel;
```
Os tipos primitivos do C++ e do Java são:
```
//tipos primitivos do C++
int
float
double
char
unsigned
long int
long long int
short
bool

//tipos primitivos do Java
int
float
double
char
boolean
byte
short
long
```
